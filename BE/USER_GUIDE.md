## Feature Guides

###  1. Question Bank Management

**Purpose**: Organize and manage your chemistry questions

**How to Use**:

1. **Browse Questions**
   - View all questions in card format
   - See topic, difficulty, and type badges
   - Check answer options for multiple choice

2. **Search & Filter**
   - Use search bar for keywords
   - Filter by topic (dropdown menu)
   - Filter by difficulty level

3. **Add New Question**
   - Click "+ Add Question" button
   - Fill in the form:
     - Select topic
     - Choose difficulty (Easy/Medium/Hard)
     - Select question type
     - Enter question text
     - Provide correct answer
   - Click "Add Question" to save

4. **Edit or Delete**
   - Click edit icon on any question card
   - Modify details and save
   - Click trash icon to delete

5. **Duplicate Questions**
   - Click copy icon to duplicate
   - Useful for creating variations

**Tips**:
- Tag questions consistently for easier filtering
- Include detailed explanations for complex topics
- Balance difficulty levels in your bank

---

###  2. AI Exercise Generator

**Purpose**: Automatically generate chemistry exercises using AI

**Step-by-Step Guide**:

1. **Configure Parameters**
   - Enter topic (e.g., "Chemical Bonding", "Stoichiometry")
   - Select difficulty level
   - Choose number of questions (3-20)
   - Add optional context for better results

2. **Generate Exercises**
   - Click "Generate Exercises"
   - Watch progress bar (takes ~3 seconds)
   - Review generated questions

3. **Review Results**
   - Each question shows:
     - Question text
     - 4 multiple choice options
     - Correct answer (highlighted in green)
     - AI-generated explanation

4. **Save or Export**
   - Click "Save to Bank" to add to question collection
   - Click "Download PDF" for printable version

**Best Practices**:
- Be specific with topic names
- Use additional context for specialized content
- Review questions before using in class
- Edit generated questions if needed

---

###  3. Exam Generator

**Purpose**: Create multiple-choice exams from your question bank

**How to Create an Exam**:

1. **Basic Settings**
   - Enter exam title
   - Set total number of questions
   - Choose number of versions (1-4)

2. **Select Topics**
   - Click on topics to include
   - Adjust question count per topic
   - Monitor total question distribution

3. **Configure Options**
   - Toggle "Randomize Questions" (recommended for multiple versions)
   - Toggle "Include Answer Key" for grading reference

4. **Generate & Preview**
   - Click "Generate Exam"
   - Review in preview modal
   - Check question distribution

5. **Export**
   - Download PDF for printing
   - Each version has unique question order
   - Answer key included if selected

**Tips**:
- Create multiple versions to prevent cheating
- Balance topics according to curriculum weight
- Always preview before printing
- Keep answer keys secure

---

###  4. OCR Grading System

**Purpose**: Automatically grade scanned answer sheets

**Grading Workflow**:

1. **Upload Answer Sheets**
   - Click upload area or drag-and-drop
   - Support formats: PDF, JPG, PNG
   - Maximum file size: 10MB

2. **Start Processing**
   - Click "Start OCR Grading"
   - Watch progress indicator
   - Processing takes ~3 seconds per sheet

3. **Review Results**
   - View class statistics:
     - Average score
     - Total students graded
     - Pass/fail distribution
   
4. **Analyze Individual Performance**
   - Click "Details" on any student
   - See question-by-question breakdown
   - Green = correct, Red = incorrect
   - Shows correct answer for wrong responses

5. **Export Grades**
   - Click "Export CSV"
   - Import into gradebook software
   - Share with students or administration

**Class Statistics Shown**:
- Average class score
- Number of students
- Total questions
- Score distribution

**Best Practices**:
- Ensure scans are clear and high contrast
- Use standard bubble sheet format
- Verify OCR accuracy on first few sheets
- Keep digital copies of original scans

###  5. Analytics & Reports

**Purpose**: Track student performance and identify trends

**What You Can See**:

1. **Overview Statistics**
   - Average class score
   - Total students across classes
   - Exams graded this period
   - Top performer highlight

2. **Performance Trends**
   - Line chart showing score progression
   - Multiple classes comparison
   - Identify improvement or decline

3. **Topic Performance**
   - Bar chart of average scores by topic
   - Identify weak areas needing review
   - Plan targeted interventions

4. **Score Distribution**
   - Pie chart of grade ranges
   - Understand class distribution
   - Identify struggling students

5. **Skills Assessment**
   - Radar chart of competency areas
   - Understanding, application, analysis
   - Problem-solving and critical thinking

6. **Top Performers**
   - Ranked list of highest scorers
   - Individual averages
   - Performance trends (improving/stable/declining)

**Using Analytics Effectively**:
- Review weekly to catch issues early
- Compare current period to previous
- Use topic data to adjust lesson plans
- Recognize and reward top performers
- Provide extra support to struggling students

## Tips & Best Practices

###  Workflow Optimization

**Daily Tasks**:
1. Check dashboard for recent exam results
2. Review pending grading queue
3. Respond to notifications

**Weekly Tasks**:
1. Generate exercises for upcoming lessons
2. Add new questions to bank
3. Review analytics for intervention needs

**Monthly Tasks**:
1. Create and finalize exams
2. Analyze long-term trends
3. Update question bank with new content

### Efficiency Tips

1. **Use Quick Actions**
   - Dashboard shortcuts save time
   - Frequently used features one click away

2. **Bulk Operations**
   - Grade multiple students at once with OCR
   - Export multiple exams as batch

3. **Template Reuse**
   - Duplicate successful questions
   - Reuse exam structures

4. **Keyboard Shortcuts**
   - Tab to navigate forms
   - Enter to submit
   - Escape to close modals

### Security Best Practices

1. **Logout After Use**
   - Click logout in sidebar
   - Especially on shared computers

2. **Secure Answer Keys**
   - Don't share with students
   - Store PDFs in secure location

3. **Verify OCR Results**
   - Spot-check automated grades
   - Report discrepancies

###  Data Management

1. **Regular Backups**
   - Export question banks monthly
   - Download important exam PDFs

2. **Organization**
   - Use consistent topic naming
   - Tag questions clearly
   - Maintain difficulty balance

3. **Quality Control**
   - Review AI-generated content
   - Test questions before using
   - Update outdated material

###  Troubleshooting

### Common Issues

**Can't Login?**
- Verify email and password
- Try demo account buttons
- Clear browser cache

**Questions Not Showing?**
- Check filter settings
- Clear search terms
- Refresh page

**OCR Not Working?**
- Verify file format (PDF, JPG, PNG)
- Check file size (<10MB)
- Ensure scan quality

**Charts Not Loading?**
- Wait for data to load
- Refresh browser
- Check internet connection
